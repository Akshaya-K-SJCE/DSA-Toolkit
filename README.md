# DSA-Toolkit
**Collection of Data Structures and Algorithms implemented in Java.**
This repository contains implementations of core Data Structures and Algorithms in Java.  
Designed as a toolkit for learning and practicing coding problems.

## Modules
### Arrays
- ReverseArray.java
- MaxElement.java

### Linked List
- Node.java
- ReverseLinkedList.java
- RotateLinkedList.java
- DetectCycle.java

### Recursion
- Fibonacci.java
- ReverseString.java

## How to Run
1. Clone the repo:     `git clone https://github.com/Akshaya-K-SJCE/DSA-Toolkit.git`
2. Navigate to folder: `cd DSA-Toolkit`
3. Compile:            `javac linkedlist/ReverseLinkedList.java`
4. Run:                `java linkedlist.ReverseLinkedList`

## Key Learning Outcomes
- Practiced arrays, linked lists, and recursion
- Applied time and space complexity analysis
- Wrote clean, modular, and well-documented code
